## Readme
Hệ thống được xây dựng nhằm mục đích tổng hợp báo cáo, theo dõi lịch họp của Hệ thống Hội nghị truyền hình trực tuyến.
"The system was developed for the purpose of aggregating reports and monitoring the meeting schedules of the Video Conferencing System."

## Run Locally
**Prerequisites:**  Node.js

1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
